===========================================
2D LID-DRIVEN CAVITY LBM SOLVER
===========================================

PROJECT GOAL:
-------------
Simulate incompressible fluid flow in a 2D lid-driven cavity using the Lattice Boltzmann Method (LBM) with CUDA acceleration and real-time OpenGL visualization.

PROJECT STRUCTURE:
------------------
├── source/           # .cpp and .cu files
├── include/          # .h and .cuh files
└── extern/           # External libraries
    ├── glad/         # OpenGL loader
    ├── glfw/         # Window/context creation
    └── glm/          # Math library

LIBRARIES USED:
---------------
- CUDA: GPU-accelerated computation
- OpenGL: Real-time visualization
- GLAD: OpenGL function loader
- GLFW: Window management and input
- GLM: Vector/matrix math

BASE SIMULATION PARAMETERS:
---------------------------
- x_resolution: Grid width (cells)
- y_resolution: Grid height (cells)
- lid_velocity: Top wall velocity (u_lid)
- time_step: dt for temporal discretization
- viscosity: Kinematic viscosity (ν)

LATTICE BOLTZMANN METHOD (D2Q9):
--------------------------------
The D2Q9 model uses 9 velocity directions in 2D:

Discrete velocities (c_i):
  6  2  5
   \ | /
  3--0--1
   / | \
  7  4  8

LBM Evolution Equations:
1. Collision (BGK): f_i* = f_i - (1/τ)(f_i - f_i^eq)
2. Streaming: f_i(x+c_i*dt, t+dt) = f_i*(x,t)

Equilibrium distribution:
f_i^eq = w_i * ρ * [1 + 3(c_i·u) + 9/2(c_i·u)² - 3/2(u·u)]

Weights (w_i):
- w_0 = 4/9 (center)
- w_1,2,3,4 = 1/9 (cardinal)
- w_5,6,7,8 = 1/36 (diagonal)

Macroscopic quantities:
- Density: ρ = Σ f_i
- Velocity: u = (1/ρ) Σ f_i * c_i

Relaxation parameter:
τ = 3ν/c² + 0.5
where c = dx/dt (lattice speed)

BOUNDARY CONDITIONS:
--------------------
- Top wall (lid): Moving wall at u_lid (velocity BC)
- Other walls: No-slip (bounce-back BC)

CUDA IMPLEMENTATION:
--------------------
- Collision step: CUDA kernel
- Streaming step: CUDA kernel
- Boundary handling: CUDA kernel
- Velocity field output: Write to vec4 array (x,y velocity + magnitude + padding)

OPENGL VISUALIZATION:
---------------------
- Display velocity field as colors or vectors
- Read vec4 array from CUDA
- Real-time rendering of flow patterns

KEY FEATURES:
-------------
- GPU-accelerated computation
- Real-time visualization
- Standard benchmark problem (lid-driven cavity)
- Modular code structure
